Basic layout of Forms

We can use the .row group and .col nested inside it to achieve horizontal row with several vertical columns. An even better practice for forms is to use .form-row instead which the columns will be less spread out and more tightly placed

Using .form-row

Forms can be aligned horizontally by customizing using the format .col-#-# where first # is size (lg, sm, md...) and second is the integer in the 12 line grid system (How many columns it take up)

Inline forms

We can easily make an inline form row using class .form-inline which will set the form to be flex. We may need to use some bootstrap shorthand Spacing classes to adjust the spacing.
