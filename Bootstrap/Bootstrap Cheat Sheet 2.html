Carousel

  • In Bootstrap, it is easier to create a Carousel compared to doing it by ourselves. The carousel involves both the Bootstrap's CSS classes and Javascript
  • First of all, the general div must be applied the classes:
    • carousel
    • slide
    as well as the data-ride attribute: data-ride='carousel'
  • The indicators of the carousel are made by an ordered list. In the ordered list the class carousel-indicators must be applied.
    In the ordered list, it will contain list items in the amount of the actual carousel slides. Each list item must be applied
    • data-target='#YOURID'
    • data-slide-to='#YOURNUM'
    Where the #YOURID is the ID of the general div for carousel, and #YOURNUM is the slide number starting from 0. Additionally, the default active indicator shall be applied a class of active
  • Then comes the actual content in the carousel. The general div for carousel slides shall be applied a class of carousel-inner and all the carousel items shall be having a class of carousel-item and the default active slide shall be applied the active class
  • As for the carousel controllers, they are wrapped in an anchor tag. The href must be set to the ID of the carousel, and some classes and data to be applied are:
    • carousel-control-prev
    • carousel-control-next
    • role='button'
    • data-slide='prev'
    • data-slide='next'
    and in the anchor tag, we have a span which shows the button logo itself, with class applied carousel-control-next-icon carousel-control-prev-icon

Collapsible Content

  • Collapsible content section usually consists of a collapsible div and a toogle, either a anchor tag (link) or a button for toggling whether the collapsible div is shown or not. This is done by Bootstrap through transforming the height property of the element.
  • For toggle using anchor tag, it must have its href attribute set to the ID of the collapsible div, and set the data attribute: data-toggle='collapse'
  • As for toggle using button, we don't have the href attribute, therefore we have to set an additional data attribute: data-toggle='collapse' data-targt='#ID'
  • As for the collapsible div itself, we just have to set the unique ID for the collapsible div so that it can be used in the toggler, as well as apply the class collapse

Dropdown Menu

  • A dropdown menu consists of a button for toggling the dropdown, as well as the actual dropdown menu itself. All of these must be wrapped inside a general div for Bootstrap
  • For the general div, we need to apply a Bootstrap class of dropdown
  • The button can be implemented using an anchor tag. We can ignore about the href attribute, since Bootstrap doesn't use it. For the classes to be applied: dropdown-toggle and the data attributes: data-toggle='dropdown'
  • The actual dropdown menu is a div containing items. For the wrapper div, apply the class dropdown-menu while for the child items, apply dropdown-item

  • A base nav is just a very simple navigation menu - A row of clickable links. Basically it consists of an unordered list with Bootstrap class applied: nav
  • Within the unordered list consists of list items which have the classes applied: nav-item
  • If the list item consists of anchor tag (link), apply the class nav-link

Navigation Bar (NavBar)

  • A Navbar is the horizontal navigation menu usually located on top of a webpage. It typically consists of a brand logo, a burger menu (for mobile users) and the nav menu consisting of nav links (Using collapsible div)
  • The base container div (use nav) will need to apply the classes navbar For the navbar to be expanded (Expand the nav menu), use navbar-expand-lg
  • The brand logo (Usually clickable - use anchor tag) will have the class of navbar-brand
  • The burger menu is a toggler for the nav menu in mobile devices. In the button element, apply the following classes: navbar-toggler along with the following data attributes: data-toggle='collapse' data-target='#IDHERE'
    Inside it we will have a span element to represent the logo, with class navbar-toggler-icon
  • Now the actual collapsible div will need a ID applied to be used for burger button. Also, apply those classes: collapse nav-bar-collapse
  • In the collapsible div, it shall contain elements, like a base navigation menu. However, instead of using class nav, use navbar-nav
