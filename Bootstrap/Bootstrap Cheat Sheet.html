Bootstrap Responsive Breakpoints

  • Bootstrap is a mobile-first framework. This means that all the styling classes that we've applied are targeted for mobile devices first.
  • Although it is not obvious at first glance, we can obviously see the designing pattern when we apply for some responsive breakpoints. Responsive breakpoints are the certain sizes that we can apply to certain styles, so that they poccess different styling on different viewport sizes.
  • When responsive breakpoints are applied, what it does is basically apply the media queries for the very same styles. Like so:
    @media (min-width: 576px) { ... }
Breakpoints Screen size Description
xs <576px No actual media query for this. It is the default for all Bootstrap styles
sm >576px Phones in landscape mode
md >768px Tablets
lg >992px Desktops
xl >1200px Large Desktops

Those rectangles demonstrate responsive breakpoints: When the device width exceeds the responsive breakpoint, it will expand to cover 100% width

Margins

  • Bootstrap comes with some easy ways to set margins on an HTML element. It provides a scale in which we can set the margin to. The scale ranges from 0 to 5. which scales from 0px, 4px up to 48px maximum. See:
Margin Type Classes Description
Margin all around m-0 m-1 m-2 m-3 m-4 m-5 Margin on all sides (left, right, top, bottom)
Margin left ml-0 ml-1 ml-2 ml-3 ml-4 ml-5 Margin on left side
Margin right mr-0 mr-1 mr-2 mr-3 mr-4 mr-5 Margin on right side
Margin top mt-0 mt-1 mt-2 mt-3 mt-4 mt-5 Margin on upper side
Margin bottom mb-0 mb-1 mb-2 mb-3 mb-4 mb-5 Margin on bottom side
Margin Horizontal mx-0 mx-1 mx-2 mx-3 mx-4 mx-5 Margin on left and right
Margin Vertical my-0 my-1 my-2 my-3 my-4 my-5 Margin on top and bottom
Margin Auto m-auto Automatically apply the margin to elements which width is smaller than 100% the parent container size, so that it centers
Breakpoint Variants m()-(size)-(1-5) Apply the respective margin size according to the responsive breakpoints

Bootstrap Grid System

  • Bootstrap follows a design pattern for grid system: Every row shall be wrapped in a row container. The row container shall be applied the class: row
  • Every row is lined from 1 to 12. The mix and match of the column elements must add up to 12. Every column item shall be applied their respective sized column classes: col-{} where {} represents scale from 1 to 12.
