JavaScript Vanilla BOM Cheat Sheet

About BOM

Like DOM, BOM, or Browser Object Model allows the Java script to interact with the browser itself.

The highest level of the BOM is window object. It represents the browser window itself and all the javascript functions, objects and variables belong to it. Even the DOM is actually part of the BOM, which is accessed by window.document. However in most cases, the window keyword can be ignored, that's why we can refer to DOM using document and not its full path document.window

Some of the BOM methods:
  • window.innerHeight returns inner height of browser window
  • window.innerWidth returns inner width of browser window
  • window.open() opens a new window. Optional argument is a URL indicating where the new window should go
  • window.close() closes the current window
  • window.moveTo(x, y) moves the selected window. We can use it on a window variable returned with window.open. Then, we can make it to be focused using window.focus. window.moveBy(x, y) also works but it moves relative to the window's position
  • window.resizeTo(x, y) resize the current window. window.resizeBy(x, y) works similar but relative to the window's size
  • window.alert(msg) pops an alert box along with a message
  • window.confirm(msg) pops an confirm box with given message. This method will have a return value of either true if user clicks OK, or false if cancel is clicked. This can be used in a if... else block for deciding actions.

  • window.prompt(msg, defaultValue) pops an prompt box to obtain input from user. This method returns the value of input if user clicks OK, and null if user clicks cancel.

  • window.setTimeout(function, millisec) and window.clearTimeout(setTimeoutObject) work together to start and stop a timer to execute a function. .setTimeout works such a way that after specified millisecs of where the code is executed, will call upon the function unless clearTimeout is called with the setTimeout variable passed in as argument

Window Screen Object

The window.screen object contains information about the user's screen.

Some of the window.screen properties:
  • screen.width returns the width of visitor's screen
  • screen.height returns the height of visitor's screen
  • screen.availWidth returns the available width of visitor's screen, which excludes the interface features like Windows Taskbar
  • screen.availHeight returns the available height of visitor's screen, which excludes interface features like Windows Taskbar
  • screen.colorDepth or screen.pixelDepth returns the number of bits used to display one color

Window Location Object

window.location object represents the current page location, which can be used to get the current page address (URL), and even redirect the browser to a new page.

Some of window.location properties and methods:
  • window.location.href returns the href (URL) of the current page
  • window.location.hostname returns the domain name of the web host (Eg: www.google.com)
  • window.location.pathname returns the path, and filename of the current page (The part of URL after domain name)
  • window.location.protocol returns the web protocol of the page (Eg: HTTP/ HTTPS... )
  • window.location.port returns the internet host port (Browsers may not display default port number of 80(http) or 443(https) )
  • window.location.assign(URL) loads a new document from given URL argument

Window History Object

window.history object contains the browser's history.

Some of the window.history methods:
  • history.back() is same as clicking back in the browser
  • history.forward() is same as clicking forward in the browser

Window Navigator Object

window.navigator object contains information regarding the visitor's browser

Some of the window.navigator properties and methods:
  • navigator.cookieEnabled returns if the cookie is enabled in browser
  • navigator.appName returns the application name of browser. However most popular browser name will return 'Netscape'
  • navigator.appCodeName returns the application code name of browser. However most popular browser name will return 'Mozilla'
  • navigator.product returns the product name of browser engine. However most popular browser will return 'Gecko'
  • navigator.appVersion returns the version information of browser
  • navigator.userAgent returns the user-agent header send by the browser
  • navigator.platform returns the browser platform (OS that it runs)
  • navigator.language returns the browser's language
  • navigator.onLine returns if the browser is online or not
  • navigator.javaEnabled() method returns if Java is enabled or not
