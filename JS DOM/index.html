JavaScript Vanilla DOM Manipulation Cheat Sheet

About DOM

In DOM (Document Object Model), all HTML elements are defined as objects. The highest of them all, document object represents our web page. Using JS, we can manipulate everything that occurs in a web page

It's important to note that the text or contents that we type in between the tags, goes into the Text Node of the element object. See figure below:

Document Object

The document object is the highest element in the DOM. It represents the web page, and carries informations that we may need

Some of the document properties:
  • document.domain to obtain the domain of the webpage
  • document.URL to obtain the full URL of the webpage
  • document.title to obtain the title of the webpage (in title tag)
  • document.doctype to obtain the doctype of the webpage
  • document.head to get the head element of this document
  • document.body to get the body element of this document
  • document.all to get ALL the elements of this document
  • document.forms to get all the forms of this document
  • document.links to get all the links (anchor tag or areas with href) of this document
  • document.images to get all the images of this document
and many more...

Selectors

There are mainly 4 ways of selecting elements in the document, namely,

  • document.getElementById to get element by ID (Remember every element can only have ONE unique ID
  • document.getElementsByClassName to get element(s) by class name. It returns an array-like object which we can access by index number and using for loop
  • document.getElementsByTagName to get element(s) by tag name. It returns an array-like object which we can access by index number and using for loop
  • document.querySelector to get element by using CSS Selectors. It returns the first element matching the CSS selectors only
  • document.querySelectorAll to get element(s) by using CSS Selectors. It returns an array-like object which we can access by index number and using for loop

HTML Text and Style

To get the contents wrapped inside the tags, we can use those: .innerText, .textContent and .innerHTML

Both .innerText and .textContent returns the text contained directly and indirectly (wrapped in other child node/ tags) of the selected element. The difference is that .textContent will return text content as it is, with extra spaces just like in HTML. Also, .textContent won't ignore text inside <script> and <style> tags, as well as text that are hidden with CSS rules (like display: none)

.innerHTML will return all the text wrapped inside the selected element, including the tags.

We can assign text to the .innerText and .textContent, both of which will remove all child nodes and append a text node containing the specified string (Tags are ignored). Setting .innerHTML will make the HTML tag to be actually rendered instead

We can set attributes of element using either of those: element.attribute = value where attribute is any of attribute properties, like href, or element.setAttribute(attribute, value)

We can change the CSS style of element by using element.style.properties = value where properties is any CSS property like background-color, and value being its respective valid value

Traversing in the DOM

There are usually 2 variants: Node and Element. Note that text Node is considered a node as well. Therefore if between the tags there is white spaces, using node may return text node instead, which is out of expectation

To get the parent node/element of an element, we can use .parentNode or .parentElement.

To get the child node(s)/ element(s) of an element, we use .childNodes or .children.

To get the first child, use .firstChild or .firstElementChild.

To get the last child, use .lastChild or .lastElementChild.

To get the next sibling, use .nextSibling or .nextELementSibling.

To get the previous sibling, use .previousSibling or .previousElementSibling.

Creation and Destruction of Element

To create a new element, we use document.createElement(tag) where tag is the name of the tag. Usually this expression is assigned to variable and used later

To add text into the element, we would need to create a Text Node, via document.createTextNode(txt) where txt is a string type argument. After creation, it has to be added to the element as child node (see below).

To change the class and id of the element, we can go ahead and use assignment operator, elem.className = 'classname' for setting classes and elem.id = 'id' for setting id

To perform operations on element such as appending, removing and replacing, use elem.appendChild(e), elem.removeChild(e) and elem.replaceChild(new, old)

Event Listeners

We add event listeners to selected element using .addEventListener(type, function) and remove it by .removeEventListener(type, function)

  • click
  • keydown
  • keyup
  • keypress
  • load fired when this element loads
  • mousedown
  • mouseup
  • mouseenter fired when pointer enters the area
  • mouseleave fires when pointer leaves the area
  • mouseover fires when pointer enters the inner content
  • mouseout fires when pointer leaves the inner content
  • focus
  • blur
  • input
  • change
